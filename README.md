# Synaptic Code

Multi-Model AI Agent Orchestration Plugin for OpenCode.

## Overview

Synaptic Code is a powerful plugin that extends OpenCode with multi-agent orchestration, parallel background agents, and crafted LSP/AST tools.

## Installation

```bash
npm install synaptic-code
```

## Features

- **Multi-Agent Orchestration**: Sisyphus orchestrates Hephaestus, Oracle, Librarian, Explore
- **Parallel Execution**: Fire multiple specialists in parallel
- **LSP & AST Tools**: IDE precision for agents
- **Built-in MCPs**: Web search, official docs, GitHub code search
- **Hash-Anchored Edits**: Zero stale-line errors
- **Tmux Integration**: Full interactive terminal support

## Agents

| Agent | Role |
|-------|------|
| Sisyphus | Main orchestrator |
| Hephaestus | Autonomous deep worker |
| Oracle | Architecture & debugging |
| Librarian | Docs & code search |
| Explore | Fast codebase grep |
| Prometheus | Strategic planner |
| Atlas | Todo orchestration |
